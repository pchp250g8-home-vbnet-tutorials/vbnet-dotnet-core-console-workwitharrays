WorkWithArrays. VBNet Tutorials. DotNet Core Platform. Simple Input/Output. Arrays. Algorithms. Console Application.
A console application in the VB.Net programming language. 
The program does the following:
   1. Asks the user to enter the number of elements in the array.
   2. Allocates memory and creates a dynamic array with the number of elements entered previously.
   3. The first loop with the parameter is executed.
      3.1. The initial value of the parameter is set to 1, and the final value is set to the number of array elements.
      3.2. A series of loops are executed, where the array elements are filled by entering data from the keyboard.
           A message is displayed on the screen indicating which element number needs to be entered.
      3.3. The value of the loop parameter is compared with the final value. If it is less, the loop continues. Otherwise, it terminates.
      3.4. The loop parameter variable is incremented by the loop increment. In this example, it is 1.
   4. The second loop with the parameter is executed.
      4.1. The initial value of the parameter is set to 1, and the final value is set to the number of array elements.
      4.2. A series of loops are executed, where the array elements are displayed on the screen in a space-separated string.
      4.3. The value of the loop parameter is compared with the final value. If it is less, the loop continues. Otherwise, it terminates.
      4.4. The loop parameter variable is incremented by the loop step. In this example, it is 1.
Developed in the Microsoft Visual Studio integrated environment.


WorkWithArrays. Занятия по VB.Net. Платформа DotNet Core. Простой Ввод/Вывод. Массивы. Алгоритмы. Консольное приложение.
Консольное приложение написанное на языке программирования VB.Net. 
Программа делает следующее:
   1. Просит пользователя ввести количество элементов в массиве.
   2. Выделяет память и создаёт динамический массив с количеством элементов введённым ранее.
   3. Выполняется первый цикл с параметром.
      3.1. Начальное значение параметра устанавливается равным 1, а конечное - числу элементов массива.
      3.2. Выполняется серия цикла, где заполняются элементы массива путём ввода с клавиатуры.
           На экране выводится сообщение о том, какой по номеру элемент нужно вводить.
      3.3. Значение параметра цикла сравнивается с конечным. Если оно меньше цикл выполняется. В противном случае - завершается.
      3.4. Переменная параметра цикла увеличивается на шаг цикла. В данном примере он равен 1.
   4. Выполняется второй цикл с параметром.
      4.1. Начальное значение параметра уставливается равным 1, а конечное - числу элементов массива.
      4.2. Выполняется серия цикла, где выводятся элементы массива на экран в строку через пробел.
      4.3. Значение параметра цикла сравнивается с конечным. Если оно меньше цикл выполняется. В противном случае - завершается.
      4.4. Переменная параметра цикла увеличивается на шаг цикла. В данном примере он равен 1.
Разработано в интегрированной среде Microsoft Visual Studio.
